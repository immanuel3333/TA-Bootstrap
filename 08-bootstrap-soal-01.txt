1. Apa itu CSS Framework
CSS Framework adalah kerangka kerja yang fungsinya memudahkan pengguna dalam mendesain website atau aplikasi pada bagian layout dan tema.
CSS Framework berisi sekumpulan kode css yang sudah siap dipakai.
